# react-calculator-count-app


## 项目介绍
+ 测试鼠标速度的小游戏
+ 基于```react```, ```dva```, ```typescript```, ```antd```
+ 学习 ts 之后的练手小游戏


### 安装依赖
```
npm install 
```

### 启动项目
```
npm start

浏览器地址栏接上 /example 即可玩耍
```

### 注意
```
非常简单的小游戏, 但是包含了 ts 在 react 中的应用,
tslint.json 可以按需配置
```

### Enjoy
